## Projects

### Autonomous CO₂ Monitoring Robot
**Autonomy · Planning · Control · SLAM**

Designed an autonomous mobile robot capable of navigating indoor environments while mapping CO₂ concentration levels.

- Implemented autonomous path planning with real-time obstacle avoidance  
- Integrated **PID control** on Arduino for precise motion control  
- Implemented **SLAM** to generate spatial heat maps of CO₂ levels  
- Included operator-controlled fallback mode for safety  
- Tools: **C++, ROS, Arduino, SLAM, PID Control**

---

### Human–Robot Interaction in Virtual Reality
**Human-Robot Interaction · Real-Time Systems**

Developed a real-time human–robot interaction system enabling a VR-based high-five with an industrial robotic arm.

- Integrated **Unity VR** environments with **KUKA robotic arms**  
- Implemented low-latency UDP communication using multithreaded C# and Python  
- Fused **Vicon motion capture** data with VR headset tracking  
- Used **Google Protocol Buffers** for structured real-time messaging  
- Tools: **Unity, C#, Python, UDP, Vicon, Protobuf**

---

### Automated Pet Feeder
**Embedded Systems · Full-Stack Integration**

Designed and built an automated pet feeding system with web-based scheduling and custom hardware integration.

- Developed a web interface for weekly scheduling using **HTML, CSS, and Node.js**  
- Deployed system on **Raspberry Pi** with Arduino-controlled hardware  
- Designed a custom **3D-printed enclosure** integrating motors and sensors  
- Tools: **Node.js, Raspberry Pi, Arduino, OnShape**
